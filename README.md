# Automatización de aire aconcidiconado Cuarto de Pruebas INNES

Sistema de control con equipos KMC, protocolo de comunicación BACnet

## Controladores de sistema

- UMA-01 *BACnet*

> DI `10000` 
>
> MAC `4` 

- PAH-PAF-PAC-UMA02 *BACnet*

> DI `10000` 
>
> MAC `4` 

- CP-01 *BACnet*

> DI `10000` 
>
> MAC `4` 

- CP-02 *BACnet*

> DI `10000` 
>
> MAC `4`

Sistema de Control de Aire Acondicionado

Sistema de control y automatización de los equipos de Aire Acondicionado para la ambientación del Cuarto de Pruebas y Oficinas de INNES Aire.

Información Técnica

Descripción General

Sistema de control centralizado, integrado por tarjetas de control DDC, conectadas por medio de red RS-485 con protocolo de comunicación BACnet MS/TP para centralizar su Operación mediante display TouchScreen.

Introducción

Propósito: Información técnica y operativa del sistema de control de los equipos de aire Acondicionado de INNES Aire.

Alcance: Esta documentación establece los parámetros operativos del sistema, así como la documentación sobre este.

Audiencia: Esta documentación está dirigida al personal técnico encargado de la operación y mantenimiento del sistema de control

Descripción

Descripción general: El sistema de control de los equipos de aire, es un sistema de control centralizado, el cual está basado en una topología de control BACnet

Componentes: El sistema está dividido en los siguientes sistemas:

UMA-01 UMA-02 PAH PAF PAC Vigas Frías Cuarto de pruebas
